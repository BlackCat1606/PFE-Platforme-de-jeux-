
Veuillez Installer: NodeJS 7.6+

Pour exécuter le projet :
Ouvrez la ligne de commande dans le répertoire du projet et Lancez les commandes suivantes : 
1. `npm install --no-optional` (*)
2. `npm run start`
3. Browse to `http://localhost:8000`
4. You can login as `administrator`/`administrator`

> (*) Si `npm install --no-optional` a des problémes avec `sqlite3`, essayez la commande avec `--build-from-source`.
